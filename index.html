        À propos & FAQ

        Qu'est-ce que Budget Scanner ?

        Un outil minimaliste et gratuit pour comprendre rapidement où va votre argent, sans créer de compte ni partager vos données.

        Mes données quittent-elles mon navigateur ?

        Non. Tout le traitement (parsing, catégorisation, calculs, génération de PDF, sauvegarde de l'historique) se déroule localement. Le navigateur ne contacte aucun backend.

        Où est stocké l'historique ?

        Dans le localStorage de votre navigateur, uniquement sur cet appareil. Vous pouvez tout effacer à tout moment via le bouton « Tout effacer ».

        Quels formats sont supportés ?

        CSV, TSV, TXT, OFX, QIF, et PDF (extraction automatique). Vous pouvez aussi coller du texte libre tant qu'on y trouve date / libellé / montant.

        Comment fonctionne le score de santé financière ?

        Sur 100 points : 40 pour le taux d'épargne, 30 pour le ratio charges fixes / revenus, 20 pour la présence d'une épargne récurrente, 10 pour l'absence de frais bancaires.

        Code source ?

        Le projet est open-source. Voir sur GitHub →

        Confidentialité

        Tout le traitement se fait dans votre navigateur. Aucune donnée bancaire n'est transmise, stockée ou journalisée sur un serveur. Fermez l'onglet, tout disparaît (sauf l'historique que vous avez choisi de sauvegarder localement).
